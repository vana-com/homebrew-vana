# homebrew-tap

Homebrew tap for Vana CLI releases.

The current formula tracks the latest `vana-connect` canary release while the
CLI is still in active rollout.

## Install

```bash
brew install vana-com/tap/vana
```

## Upgrade

```bash
brew update
brew upgrade vana
```

## Uninstall

```bash
brew uninstall vana
```
